Soy una subclase de `SystemWindow` con 3 componentes:
- `pilaNavegacion`
- `barraNavegacion`
- `contenedorVentana`

Manejo la inicialización de un sistema, y aporto su navegabilidad.

Inicialización
En un Playground escriba lo siguiente para iniciar el navegador listo para usar:
`Navegador abrir.`
